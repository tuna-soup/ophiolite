Use these labels consistently:

- **Stable:** intended for production with only controlled change
- **Preview:** real and supported for evaluation, but still evolving
- **Experimental:** useful for exploration, not a long-lived contract yet
- **Internal:** not part of the recommended public surface

Today, most Ophiolite surfaces on this site are **Preview**.
