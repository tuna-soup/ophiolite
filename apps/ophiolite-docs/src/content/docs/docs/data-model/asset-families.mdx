Ophiolite organizes data by **asset family** rather than by raw file type alone.

Current families include:

- logs
- trajectory
- tops
- pressure observations
- drilling observations
- seismic trace-data and related runtime paths

This family model is what lets operators, eligibility, and persistence rules stay domain-aware.
