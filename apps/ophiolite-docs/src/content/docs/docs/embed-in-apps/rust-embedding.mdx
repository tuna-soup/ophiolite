---
title: Rust Embedding
description: When and why to use the Rust surface directly.
draft: false
---

Use Rust directly when you need the deepest, most exact access to Ophiolite.

That is the right choice when:

- the workflow is ahead of wrapper coverage
- you are building a desktop backend
- you need direct access to the canonical runtime types

Rust is not the default onboarding path, but it is the canonical one.
