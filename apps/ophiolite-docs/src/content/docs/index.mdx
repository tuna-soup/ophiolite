Ophiolite is a Rust-first SDK for subsurface well data. It combines:

- strong LAS/log ingestion
- optimized package storage
- a local-first `OphioliteProject` catalog
- typed asset families for trajectory, tops, pressure observations, and drilling observations
- typed compute and derived assets
- revision-aware editing workflows for logs and structured assets

Use this site to get from first import to project-level architecture quickly:

- start with [Installation](/getting-started/installation/)
- learn the core [Concepts](/concepts/overview/)
- understand [OphioliteProject](/packages-and-projects/ophiolite-project/)
- review the current [Architecture](/architecture/overview/)
