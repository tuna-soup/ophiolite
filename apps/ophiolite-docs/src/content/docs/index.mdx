Ophiolite is the platform for applications that need canonical domain contracts, local-first runtime primitives, stable app-boundary DTOs, and embeddable subsurface charts.

It started with a strong LAS and log foundation, but it is no longer a LAS-focused project. Today the platform spans:

- well and wellbore assets such as logs, trajectory, tops, pressure, and drilling observations
- shared seismic descriptors, runtime/store paths, section and gather display DTOs, and processing-facing semantics
- survey-map and time-depth boundary types used by applications and embedders
- embeddable seismic, gather, survey-map, and well-correlation charts through Ophiolite Charts
- optimized local package and project storage
- a local-first `OphioliteProject` catalog
- typed compute and derived assets
- revision-aware editing and persistence
- frontend-safe contracts for apps and SDK adapters
