The early-access launch is centered on these chart families:

- Seismic Section
- Prestack Gather
- Well Correlation Panel
- Survey Map
- Rock Physics Crossplot

These families form the first commercially supported 2D surface for the charts SDK.

## Why These Families Ship First

They already form a coherent public surface:

- shared interaction ideas
- explicit view state
- clear scientific and subsurface use cases
- enough implementation depth to support live examples and realistic documentation
