The first public examples will be rebuilt from the existing wrapper playground into polished demo pages with cleaner controls, sharper copy, and explicit links back to the documented API.

## Planned Launch Examples

- **Seismic Section:** compare heatmap and wiggle presentation, shared probe callbacks, and overlay support
- **Prestack Gather:** focused gather exploration over the same seismic interaction model
- **Well Correlation Panel:** multi-well depth alignment with explicit panel semantics
- **Survey Map:** lightweight plan-view context for wells, trajectories, and outlines
- **Rock Physics Crossplot:** dense point-cloud interaction with host-owned axis editing

## Current Runtime

The curated examples currently run from the Svelte examples app in public mode.

## Local Development

Run the examples app from the charts workspace:

```bash
bun run dev:svelte
```

To preview the bundled site shape locally from the docs app, use:

```bash
bun run dev:with-live
```

Then open the public route you want, for example:

```text
<local-site-url>/live/?mode=public#/seismic
```

## Examples Policy

The public examples surface will favor:

- realistic defaults
- a small number of purposeful controls
- explicit links to the matching docs page

It will avoid exposing the full internal playground configuration surface as the main product demonstration.
