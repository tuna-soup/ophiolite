Ophiolite Charts is the product-facing chart SDK from the Ophiolite stack. The launch scope is focused: a small number of chart families, clear documentation, and live examples that reflect the code you can actually build on.

## Launch Scope

- Svelte-first public wrappers
- JavaScript embedding guidance for current wrappers
- Early access commercial rollout
- Curated examples for launch chart families

## Current Focus

- clear data-model boundaries
- consistent chart handles and interactions
- examples that match documented APIs
- benchmark methodology that can be repeated and explained

## Use It For

- interpretation tooling
- scientific web applications
- subsurface workflows
- domain-specific dashboards that need more than commodity line and bar charts
