# @ophiolite/contracts

Generated TypeScript contracts for frontend-safe `ophiolite` DTOs.

Current contents:

- seismic gather view/probe/viewport DTOs for prestack chart integration
- well-panel request DTOs
- resolved well-panel source DTOs
- supporting row/set DTOs for logs, trajectory, tops, pressure, and drilling

Artifacts are generated from Rust DTO definitions in the `ophiolite` repo.
